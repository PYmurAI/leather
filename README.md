# レザークラフト型紙ジェネレーター

レザークラフト用の基本パーツを、寸法入力からSVG型紙として生成するブラウザ完結型ツールです。

## 実装済み

- 長方形、R付き長方形（上のみR、下のみR、全周R）
- T字パーツ
- 上底、下底、高さで指定できる直線マチ、扇形マチ
- 縫い代ラインと菱目打ち穴位置のリアルタイム表示
- 複数パーツのA4自動配置
- 1枚に収まらない場合のA4複数ページ生成
- 同じ形のパーツをまとめて追加できる個数指定
- A4縦/横の切り替え
- 保存対象、用紙サイズ、ページ数、配置座標を確認できるA4プレビュー
- 単体SVG保存、A4複数ページSVG保存
- ブラウザ内への作業途中保存と復元
- 1万円札とクレジットカードの参考サイズ表示

## 使い方

`index.html` をブラウザで開いてください。サーバは不要です。

1. パーツ種類と寸法を入力します。
2. 必要に応じて縫い代、ピッチ、縫い線、穴位置を調整します。
3. A4向きを選びます。初期値は `横 297×210mm` です。
4. 追加個数を指定して `A4ページに追加` を押すと、現在のパーツが指定数ぶんA4台紙に自動配置されます。1枚に収まらない場合は自動で次ページに送られます。
5. `単体` / `A4ページ` を切り替えて、保存されるSVGの内容を確認します。
6. `SVG保存` で表示中のSVGを書き出します。
7. 入力内容とA4配置を残したい場合は `作業を保存` を押します。次回起動時に保存済み内容が復元され、編集中に保存地点へ戻したい場合は `保存を復元` を押します。

Phase1 の確認条件である `230mm × 65mm`、`上辺R20`、`縫い代3mm`、`ピッチ4mm` は初期値として設定済みです。
